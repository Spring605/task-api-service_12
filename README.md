﻿# Task API Service - 12

基于 Node.js 和 Express 的 RESTful API 服务，提供完整的任务管理功能。

## 功能特性

- 用户注册和 JWT 认证
- 任务 CRUD 操作  
- 数据验证和错误处理
- 分页和过滤查询

## 快速开始

1. 安装依赖: 
npm install
2. 启动服务: 
npm run dev
3. 访问: http://localhost:3000
